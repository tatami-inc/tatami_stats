Add the next value in Welford's method for computing the variance. This function should be called multiple times to obtain the mean and summed of squared differences.

Template Parameters
Output_Type of the output data.
Value_Type of the input data.
Index_Type of the row/column index.
Parameters
meanCurrent value of the mean. This should be set to zero before the first call to this function.
sumsqCurrent sum of squared differences from the mean. This should be set to zero before the first call to this function.
valValue to be used in the variance calculation.
countNumber of values that have been used so far, including val (i.e., this should start at 1).

num_allNominal length of the sparse dataset, including zeros; this should be no less than num_nonzeros.

Compute the mean and variance from sparse data. This achieves faster processing by only performing summations over non-zero elements. We use Welford's method for numerical stability (and consistency with RunningSparse).

Template Parameters

Value_Type of the input data.
Index_Type of the indices.
Parameters
[in]valuePointer to an array of values for structural non-zero elements.
num_nonzeroLength of the array pointed to by value.
num_allTotal number of values in the dataset, including the zeros not in value. This should be greater than or equal to num_nonzero.
Returns
The sample mean and variance of values in the vector. This may be NaN if there are not enough (non-NaN) values in value.
