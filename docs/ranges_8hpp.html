Compute the extremes of a dense array.

Template Parameters
get_minimum_Whether to compute the minimum. If false, the maximum is computed instead.
skip_nan_Whether to check for (and skip) NaNs. If false, NaNs are assumed to be absent, and the behavior of this function with NaNs is undefined.
Value_Type of the input data.
Index_Type of the row/column indices.
Parameters
[in]ptrPointer to an array of values of length num.
numSize of the array.
Returns
The minimum or maximum value, depending on get_minimum_. If num = 0 or (if skip_nan_ = true) there are no non-NaN values, a placeholder value is returned instead that is never less than (if get_minimum_ true) or greater than (otherwise) any non-NaN value of type Value_.

Compute extremes for each element of a chosen dimension of a tatami::Matrix.

Template Parameters
skip_nan_Whether to check for (and skip) NaNs. If false, NaNs are assumed to be absent; the behavior of this function in the presence of NaNs is undefined.
Value_Type of the matrix value, should be numeric.
Index_Type of the row/column indices.
Output_Type of the output value.
Parameters
rowWhether to compute variances for the rows.
pPointer to a tatami::Matrix.
threadsNumber of threads to use.
[out]min_outPointer to an array of length equal to the number of rows (if row = true) or columns (otherwise). On output, this will contain the minimum of each row/column. Alternatively, this may be NULL, in which case the minima are not computed.
[out]max_outPointer to an array of length equal to the number of rows (if row = true) or columns (otherwise). On output, this will contain the maximum of each row/column. Alternatively, this may be NULL, in which case the maxima are not computed.
