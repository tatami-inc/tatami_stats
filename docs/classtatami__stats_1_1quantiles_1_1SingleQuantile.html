Detailed Description

template<typename Output_, typename Value_, typename Index_>
class tatami_stats::quantiles::SingleQuantile< Output_, Value_, Index_ >

Calculate a quantile from objective vectors of fixed size.

Template Parameters
Output_Floating-point type of the output quantile.
Value_Numeric type of the input values.
Index_Integer type of the objective vector's size.

A SingleQuantile instance should constructed once, given a fixed size for the objective vectors of interest and the desired quantile probability. It can then be invoked any number of times with operator() to compute quantiles from objective vectors of that size. Specifically, it will compute a type 7 quantile, consistent with the default in R's quantile function.

This class is used by apply() to compute quantiles when iterating through a non-empty tatami::Matrix with no NaNs. For objective vectors of variable or zero length, see SingleQuantileVariableLength instead.
