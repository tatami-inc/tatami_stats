Detailed Description

template<typename Output_, typename Value_, typename Size_>
class tatami_stats::quantiles::SingleQuantile< Output_, Value_, Size_ >

Calculate a single quantile from an array of known length.

Template Parameters
Output_Floating-point type of the output quantile.
Value_Numeric type of the input values.
Size_Integer type of the number of elements.

An instance of this class is constructed once given the number of elements and the desired probability. It can then be re-used to compute a quantile for any array of size equal to the specified number of elements. Specifically, it will compute a type 7 quantile, consistent with the default in R's quantile function.
