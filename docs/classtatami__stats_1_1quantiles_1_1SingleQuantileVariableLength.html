Detailed Description

template<typename Output_, typename Value_, typename Size_>
class tatami_stats::quantiles::SingleQuantileVariableLength< Output_, Value_, Size_ >

Calculate a single quantile for arrays of variable length.

Template Parameters
Output_Floating-point type of the output quantile.
Value_Numeric type of the input values.
Size_Integer type of the number of elements.

An instance of this class is constructed once given the maximum number of elements and the desired probability. It can then be re-used to compute quantiles from containers of size less than or equal to the maximum. See SingleQuantile for more details.

Returns
