Detailed Description

template<typename Output_, typename Value_, typename Index_>
class tatami_stats::quantiles::SingleQuantileVariableLength< Output_, Value_, Index_ >

Calculate a quantile from objective vectors of variable size.

Template Parameters
Output_Floating-point type of the output quantile.
Value_Numeric type of the input values.
Index_Integer type of the objective vector's size.

A SingleQuantileVariableLength instance should be constructed once, given the maximum size of the objective vectors of interest and the desired quantile probability. It can then be invoked any number of times to compute quantiles from objective vectors of size less than or equal to the maximum. See also the SingleQuantile class for more details.

This class is used by apply() to compute quantiles when iterating through a tatami::Matrix that might contain NaNs. Specifically, NaNs are shifted to the front of each array with shift_nans() and operator()() is called on the remaining non-NaN values.

Compute the desired quantile from a dense objective vector of length no greater than max_len.

If any NaNs are present in the objective vector, we recommend shifting them to the front with shift_nans(), and then calling this method with len := len - N and ptr := ptr + N where N is the number of NaNs.

This method is not thread-safe.

Parameters
lenLength of the dense objective vector. This should be no greater than max_len.
[in]ptrPointer to an array of length len. This should not contain NaN values. On output, the order of elements in [ptr, ptr + len) may be rearranged.
Returns
Quantile of the specified probability. If len = 0, NaN is returned instead.

Compute the desired quantile from a sparse objective vector of length no greater than max_len.

If any NaNs are present in the array, we recommend shifting them to the front with shift_nans(), and then calling this method with len := len - N, num := num - N and ptr := ptr + N where N is the number of NaNs.

This method is not thread-safe.

Parameters
lenLength of the sparse objective vector. This should be no greater than max_len.
numNumber of structural non-zero elements. This should be no greater than len.
[in]ptrPointer to the start of an array of length num. This is expected to contain the values of the structural non-zero elements of the sparse vector. It should not contain any NaN values. On output, the order of elements in [ptr, ptr + num) may be rearranged.
Returns
Quantile of the specified probability. If len = 0, NaN is returned instead.
